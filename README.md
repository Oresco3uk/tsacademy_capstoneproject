# tsacademy_capstoneproject
capstone project
This capstone project focuses on predicting weekly sales across multiple stores and departments using historical data from the Walmart Store Sales Forecasting Dataset. The dataset contains sales records from 45 Walmart stores, including information about holidays, economic indicators, and promotional markdowns.

The main goal of the project is to build a machine learning model capable of accurately predicting weekly sales, especially during major holiday periods when sales fluctuate significantly.
